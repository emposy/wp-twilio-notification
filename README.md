# Twilio Notification for Contact Form 7

Contact Form 7からTwilioで電話とSMSを送信するWordPressプラグイン

## インストール

1. このリポジトリをZIPでダウンロード
2. サーバーにアップロードして解凍
3. `composer install` を実行
4. WordPressでプラグインを有効化

## 設定

WordPress管理画面 → 設定 → Twilio通知

- Account SID
- Auth Token
- Twilio電話番号
- 通知先電話番号
- TwiML URL
- SMS本文
- CF7電話番号フィールド名


【手順2】Composer実行

composer install


※ vendor/ディレクトリが作成されます
